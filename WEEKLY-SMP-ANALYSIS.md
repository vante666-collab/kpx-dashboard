# 주간 SMP 변화원인 분석 — 체크리스트 / 템플릿

매주(또는 두 날짜 비교) SMP 변화 원인을 빠르게 분석하기 위한 루틴.
도구: `smp.html` (분석 화면) + `smp_ingest.py` (원본파일 → smp_data.json 변환).

---

## 1. 파일 모으기 (KPX/EPSIS에서 다운로드)

기준일·비교일 **두 날짜** 각각에 대해 받으면 가장 강력합니다.

| # | 파일 (파일명 키워드) | 형식 | 담는 정보 | 필수 |
|---|---|---|---|---|
| 1 | `금일수요예측 목록_YYMMDD` | xlsx | 시간별 수요(=순부하) | ★ |
| 2 | `일간 계통한계가격 목록_YYMMDD` | xlsx | 시간별 SMP | ★ |
| 3 | `가격결정발전기 목록_YYMMDD` | xlsx | 시간별 한계발전기(육지) | ★ |
| 4 | `일별 수요 SMP 목록` | xlsx | 여러 날 일평균 수요·SMP | ○ |
| 5 | `제약검토서-YYMMDD..._하루전발전계획용` | hwp | 융통한계·상시/휴전 제약·제약발전 | ★ |
| 6 | `발전소 공급능력표 YYYY년 M월 D일` | txt | 연료원별 공급능력·호기별 정지/제약 | ★ |
| 7 | (선택) 하루전발전계획 가격결정자격(PSI/SMF) | - | 제약발전 가격결정 제외 확정자료 | △ |

> 전부 **한 폴더**에 모읍니다. 6은 있으면 인과 100% 입증용(없어도 정황분석 가능).

## 2. 변환 (한 줄)

```powershell
python smp_ingest.py "C:\경로\그_폴더"
```
- 결과: `smp.html` 옆에 `smp_data.json` 생성.
- 자동 추출: 일평균·시간별 SMP/순부하·시간별 한계발전기·융통전력 한계량·**상시/휴전 제약발전기 목록**.

## 3. 보기

`smp.html` 새로고침 → 맨 위 **종합 리포트**가 그 데이터로 자동 작성됨.
(파일이 없으면 내장 6/1·6/8 예시로 동작)

## 4. 리포트 읽는 순서 (인과 사슬)

1. **① 요인 분해** — 수요로 얼마, 잔차 얼마?
2. **② 가격결정발전기** — 시간대 분해(주간/저녁), 석탄·LNG 한계시간, 한계가격 곡선
3. **②-b 제약발전 가격결정자격 전환** — 제약발전이 기준일엔 가격결정(비구속)→비교일엔 제외(구속)?
4. **③ 부하–가격 디커플링** — 순부하는 비슷한데 SMP만 벌어지면 = 공급측
5. **④ 송전제약** — 융통전력 한계량 완화/강화, 휴전제약 발전기
6. **🔎 결론** — 위를 종합

## 5. 해석 가이드 (자주 나오는 패턴)

- **낮만 SMP 폭락 + 순부하 거의 안 변함** → 공급측 merit-order(태양광/저가공급↑) or 제약완화.
- **제약발전(동탄열병합·포천 등)이 비교일 낮에 가격결정에서 빠짐** → 제약 구속(유효)으로 SMF=0 → 더 싼 발전기가 한계 → SMP 하락폭 확대 (제2.4.2조⑤8호나).
- **융통전력 한계량 완화(+MW)** → 수도권 저가전력 유입↑ → 비싼 제약발전 의존↓ (현행규칙상 하루전발전계획 통해 SMP에 반영).
- **저녁은 변화 작은데 낮만 큼** → 수요(부하) 요인 아님(저녁 부하가 더 큼). 낮 특이요인(태양광·제약) 의심.

## 6. 한계 / 주의

- **.hwp는 브라우저가 못 읽음** → 반드시 `smp_ingest.py`로 변환.
- **휴전제약은 날짜마다 바뀜** → 매번 검토서를 넣어 재추출(자동).
- **인과는 정황 수준** — 가격결정자격(PSI/SMF) 자료가 있어야 확정.
- 연료단가·신재생비중·원전가동률은 무료 일별이 빈약 → 표에서 수동 보정 가능(잔차로 남김).
- `smp.html` 수정 시 배포본 `_site/smp.html`에도 복사.
